## 四、架构概览

```
┌─────────────────────────────────────────────────────────┐
│  前端 (Vue + AG Grid + dockview-vue)                     │
│  - SQL 编辑器 / 结果集浏览器 / 文件导入                   │
└──────────────────┬──────────────────────────────────────┘
                   │ Tauri Invoke (Arrow)
┌──────────────────▼──────────────────────────────────────┐
│  DBI (统一数据访问入口)                                   │
│  ├── DBI::query() / execute()                           │
│  ├── Session (会话管理)                                  │
│  └── QueryRouter (智能路由)                              │
├─────────────────────────────────────────────────────────┤
│  执行引擎                                                │
│  ├── DriverEngine (原生驱动)                             │
│  ├── DuckDBEngine (本地加速/联邦查询)                     │
│  └── StreamEngine (流式处理)                             │
├─────────────────────────────────────────────────────────┤
│  数据库驱动层                                            │
│  ├── MySQL (sqlx) / PostgreSQL (sqlx)                   │
│  ├── SQLite (rusqlite) / DuckDB (duckdb-rs)            │
│  └── 智能连接池 (SmartPool)                              │
├─────────────────────────────────────────────────────────┤
│  持久化层                                                │
│  ├── SQLite (meta/project.db) - 事务性数据               │
│  ├── DuckDB (analytics/data.duckdb) - 分析性数据         │
│  └── JSON (config/*.json) - 配置数据                     │
└─────────────────────────────────────────────────────────┘
```

## 五、关键约束

| 约束              | 说明                        |
| ----------------- | --------------------------- |
| ❌ 禁止 unwrap()  | 生产代码必须使用 ? 或 match |
| ✅ 必须 CoreError | 统一错误处理                |
| ✅ Arrow 传输     | 所有数据流转使用 Arrow 格式 |
| ✅ DuckDB 只读    | 加速模式下外部数据库只读    |
| ✅ 绝对路径       | 文件数据源使用绝对路径      |
| ✅ 用户自定义名称 | 结果集支持用户自定义命名    |
| ✅ 无自动持久化   | 用户手动触发持久化          |
| ✅ 编译通过       | `cargo check` 无错误        |
