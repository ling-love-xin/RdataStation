# 分析资源管理器 前端设计方案

## 一、组件架构

```
views/workbench/
└── components/
    └── AnalyticsResourceManager/
        ├── AnalyticsResourceManager.vue          # 主容器
        ├── ResourceSearchBar.vue                # 搜索栏
        ├── ResourceTagFilter.vue                # 标签筛选栏

        │   ├── FolderItem.vue                   # 文件夹项（可折叠）
```

## 三、标签栏设计

### 3.1 标签分组（互斥）

```
┌─────────────────────────────────────────────────────────────┐
│  作用域: [全部] [🌍全局] [📂项目] [📌临时]                  │
│  类型:   [📄文件] [📊表] [🔌连接]                           │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 标签视觉设计

| 标签 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| 全部 | — | 默认 | 显示所有资源 |
| 🌍 全局 | 🌍 | 蓝色 #165DFF | 应用级资产 |
| 📂 项目 | 📂 | 绿色 #00B42A | 项目级资产 |
| 📌 临时 | 📌 | 橙色 #FF7D00 | 会话级，注意清理 |
| 📄 文件 | 📄 | 灰色 | — |
| 📊 表 | 📊 | 灰色 | — |
| 🔌 连接 | 🔌 | 灰色 | — |

### 3.3 标签交互

- **单选互斥**：同组内只能选一个
- **跨组叠加**：作用域 + 类型可同时选（如：全局 + 表）
- **清除**：点击"全部"或再次点击已选标签

## 四、资源列表设计

### 4.1 资源项视图

```
┌───────────────────────────────────────────────────────────────────┐
│ [类型图标]  资源名称                        [作用域]    [行数/大小] │
│                                                                   │
│ 示例：                                                             │
│ 📄 订单数据.csv                            [📂项目]      12.5 MB    │
│ 📊 供应商主数据 v3                        [🌍全局]      45,678 行  │
│ 📊 temp.result_123                        [📌临时]       1,234 行 │
│ 🔌 MySQL生产库                            [🌍全局]                     │
└───────────────────────────────────────────────────────────────────┘
```

### 4.2 资源项信息层级

```
第一行：类型图标 + 资源名称（主名称，可显示别名）
第二行：引用名 / 文件路径（次要信息）
右侧：作用域标签 + 统计信息（行数/大小/更新时间）
```

### 4.3 排序支持

点击列头排序：
- 名称（A-Z / Z-A）
- 作用域（全局 > 项目 > 临时）
- 大小/行数
- 更新时间

## 十七、核心设计原则

| 原则 | 体现 |
|------|------|
| **少概念，多直觉** | 标签代替层级路径 |
| **视图与存储分离** | 文件夹是视图，不是存储位置 |
| **渐进式复杂性** | 基础用户用标签，高级用户用文件夹 |
| **零心智负担** | 双击插入引用片段 |
| **数据安全** | 回收站、删除确认、依赖检查 |
